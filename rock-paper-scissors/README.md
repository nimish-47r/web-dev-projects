# ✊✋✌ Rock Paper Scissors

A simple browser-based Rock Paper Scissors game where you play against the computer.

## Features

- Classic Rock, Paper, Scissors gameplay against the computer
- Live score tracking for you and the computer
- Round-by-round history with the winning choice highlighted

## Getting Started

1. Clone the repo:
   ```bash
   git clone https://github.com/nimish-47r/web-dev-projects.git
   ```
2. Navigate to the project folder:
   ```bash
   cd web-dev-projects/rock-paper-scissors
   ```
3. Open `index.html` in your browser.

## How to Play

1. Click Rock, Paper, or Scissors.
2. The computer makes its own random selection.
3. Winner is decided by classic rules: Rock beats Scissors, Paper beats Rock, Scissors beats Paper.
4. Scores update automatically, and each round is added to the history below.
